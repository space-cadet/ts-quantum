Function wigner3j

  • wigner3j(j1, j2, j3, m1, m2, m3): Complex

  • Calculates Wigner 3j symbol using Clebsch-Gordan coefficients

    The Wigner 3j symbol is related to Clebsch-Gordan coefficients by: (j1 j2 j3) = (-1)^(j1-j2-m3) / sqrt(2*j3+1) * ⟨j1,m1;j2,m2|j3,-m3⟩ (m1 m2 m3)

    Based on verified formula from Sage/SymPy documentation: ⟨j₁ m₁ j₂ m₂ | j₃ m₃⟩ = (-1)^(j₁-j₂+m₃) * √(2j₃+1) * Wigner3j(j₁, j₂, j₃, m₁, m₂, -m₃)

    Parameters

    • j1: number

      First angular momentum

      Second angular momentum

    • j3: number

      Third angular momentum

    • m1: number

      First magnetic quantum number
